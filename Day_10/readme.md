# Python Strings
--Strings in python are surrounded by either single quotation marks, or double quotation marks.
  'hello' is the same as "hello". We can display a string literal with the print() function.
  print("HELLO")

# Some Useful opteration on Strings
  - Slicing - Specify the start index and the end index, separated by a colon, to return a part of the string(::).4
  - Upper Case - The upper() method returns the string in upper case.
  - Lower Case - The lower() method returns the string in lower case.
  - Remove Whitespace - The strip() method removes any whitespace from the beginning or the end.
  - Replace String - The replace() method replaces a string with another string.
  - Split String - The split() method splits the string into substrings if it finds instances of the separator.
  - String Concatenation - To concatenate, or combine, two strings you can use the + operator.
