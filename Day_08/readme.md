# TUPLE
A tuple is a built-in collection data type used to store multiple items in a single variable. It is ordered, immutable, and allows duplicate values.

1. Creating a Tuple
   Tuples are written with round brackets ().

# SET
A set is a built-in data type used to store an unordered collection of unique items. Sets automatically eliminate duplicate values and offer highly efficient O(1) time complexity for membership testing because they are implemented using hash tables.
Sets are written with curly brackets.
